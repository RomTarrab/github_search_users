## Installation:

<br/>

### 1. Clone the repository

```bash
 git clone https://github.com/RomTarrab/github_search_users
```
<br/>

### 2. Run this commands to install the dependencies:

```bash
 cd client
```

```bash
 npm install
```
---
```bash
 cd server
```

```bash
 npm install
```
<br/>

---
### 3. Run this command in both client and server:

```bash
 npm run dev
```
